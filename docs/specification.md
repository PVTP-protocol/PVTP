# PVTP Specification

## 1. Purpose
PVTP defines a progressive method for transferring visual content.

## 2. Goals
- faster perceived loading
- adaptive quality
- efficient transfer
- open implementation
- future extension

## 3. Transport
The protocol may be implemented over existing secure transports.

## 4. Core behavior
- request resource
- receive metadata
- render preview
- receive higher-quality layers
- finalize and cache

## 5. Versioning
The protocol should use explicit version identifiers.

## 6. Security
Authentication, integrity, and transport security should be defined clearly.

## 7. Implementation rule
The specification should remain the source of truth for all clients and servers.

# PVTP

**Progressive Visual Transfer Protocol**

PVTP is an open protocol concept for fast, adaptive, progressive delivery of visual content.

It is designed to:
- show a preview quickly,
- improve quality progressively,
- reduce wasted bandwidth,
- and provide a clean foundation for future visual transfer systems.

Start with the specification in `docs/specification.md`.
